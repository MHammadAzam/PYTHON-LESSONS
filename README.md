# Python Lessons

A structured collection of Python lessons, examples, and practice exercises designed to help beginners build a strong foundation in programming with Python.

## Topics Covered

* Python Syntax and Variables
* Data Types
* Input and Output
* Operators
* Conditional Statements (`if`, `elif`, `else`)
* Loops (`for`, `while`)
* Functions
* Lists, Tuples, Sets, and Dictionaries
* String Manipulation
* File Handling
* Exception Handling
* Object-Oriented Programming (OOP)
* Modules and Packages
* Basic Problem Solving Exercises

## Purpose

This repository was created to document my Python learning journey and serve as a reference for future projects in software development, data science, machine learning, and artificial intelligence.

## Requirements

* Python 3.x installed on your system
* A code editor such as VS Code or PyCharm

## How to Run

1. Clone the repository:

   ```bash
   git clone <repository-url>
   ```

2. Navigate to the project directory:

   ```bash
   cd Python-Lessons
   ```

3. Run any Python file:

   ```bash
   python filename.py
   ```
